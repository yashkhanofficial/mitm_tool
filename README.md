# ⚡ MITM-Pro-2026

A specialized Man-in-the-Middle tool designed for **Termux** and **Kali NetHunter**. Built for educational purposes in Ethical Hacking.

## 🛠 Features
- **ARP Spoofing:** Cut or intercept target traffic.
- **Auto IP-Forwarding:** Handles Linux kernel settings automatically.
- **Network Restore:** Safely cleans up ARP tables after the attack.
